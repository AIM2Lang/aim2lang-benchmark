README.md (mục bắt buộc)

AIM-Bench overview

Mô tả ngắn: benchmark cho adaptive learning dựa trên AIM-FRAME (goal-centric).

Data

Tóm tắt aims.jsonl, lessons.jsonl, profiles.jsonl (+ ontology.ttl nếu có).

Tasks & Metrics

Task 1: Goal→AIM — Accuracy, Recall@k, MRR

Task 2: Path Planning — PathAcc, PrereqSatisfaction, EditDistance

Task 3: Lesson Selection — Acc@1, MRR/NDCG (nếu nhiều lesson đúng)

Task 4: Profile Update — Acc, F1, Cohen’s Kappa

File formats

Mô tả từng field + ví dụ JSONL như trên.

Evaluation

Cách dùng script trong metrics/ (hoặc công thức nếu bạn không muốn release script).

Lệnh mẫu:

python metrics/compute_retrieval.py --gold benchmark/task1_goal2aim.jsonl --pred runs/task1_pred.jsonl
